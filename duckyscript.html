Sim, o código que você forneceu parece ser um exemplo de um script Duckyscript. Esse script em particular abre o Bloco de Notas (notepad.exe) e digita "Hello World!". O comando "GUI r" pressiona a tecla do logotipo do Windows + R para abrir a caixa de diálogo Executar. Em seguida, o script digita "notepad.exe" na caixa de diálogo Executar e pressiona Enter para abrir o Bloco de Notas. Depois de um atraso de 1 segundo, o script digita "Hello World!" no Bloco de Notas. Este é um exemplo simples de como um script Duckyscript pode automatizar ações em um computador.

GUI r
DELAY 500
STRING notepad.exe string/pdf/2024
ENTER
DELAY 1000
STRING Hello World!
string document-2024

Para armazenar e executar um script Duckyscript em um dispositivo USB, você precisará de um dispositivo Rubber Ducky ou um dispositivo compatível com Duckyscript. Aqui estão os passos gerais para fazer isso:

Abra um editor de texto, como o Bloco de Notas, e cole o código Duckyscript que você deseja usar. No seu caso, o código seria algo assim:
Code


GUI r
DELAY 500
STRING notepad.exe string/pdf/2024
ENTER
DELAY 1000
STRING Hello World!
STRING document-2024


Salve o arquivo com a extensão ".txt", por exemplo, "meuscript.txt".
Renomeie o arquivo para ter a extensão ".ducky", por exemplo, "meuscript.ducky".
Copie o arquivo "meuscript.ducky" para a raiz do seu dispositivo USB.
Insira o dispositivo USB em um dispositivo Rubber Ducky ou um dispositivo compatível com Duckyscript.
Quando o dispositivo USB for conectado a um computador e o script for executado, ele irá emular as ações especificadas no código Duckyscript. Certifique-se de que o dispositivo USB que você está usando é compatível com Duckyscript e que você está ciente das possíveis implicações de segurança ao executar scripts automatizados em um computador.
